Instructions d'Utilisation

Chaque composant est un fichier HTML autonome contenant :

  • Les styles CSS complets avec media queries
  • Un exemple d'utilisation
  • Une description détaillée du composant
  • La liste des fonctionnalités incluses

Pour utiliser un composant dans une page :

  1. Copier les styles CSS du composant
  2. Adapter le HTML selon les besoins
  3. Vérifier la compatibilité avec les autres composants
