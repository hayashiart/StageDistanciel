Composant - Cartes Administratives

Ce composant définit les cartes pour les niveaux administratifs, utilisé dans page1.

Il inclut :

  • Cartes avec images circulaires
  • Bordures bleues et ombres portées
  • Effet hover avec élévation
  • Disposition responsive avec flexbox
  • Images avec bordures colorées
