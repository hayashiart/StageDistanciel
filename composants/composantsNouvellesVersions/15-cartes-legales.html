Composant - Cartes Légales

Ce composant définit les cartes légales utilisées dans page2 pour présenter les informations juridiques.

Il inclut :

  • Blocs légaux avec bordures bleues et ombres
  • En-têtes avec titres et sous-titres stylisés
  • Contenu structuré avec paragraphes et listes
  • Effet hover avec élévation
  • Disposition responsive avec flexbox

Code du Travail

Législation principale

Le Code du Travail régit les relations entre employeurs et salariés.

  • Contrats de travail
  • Durée du travail
  • Congés et repos

Convention Collective

Accord sectoriel

La convention collective complète le Code du Travail pour un secteur donné.

  • Salaires minima
  • Conditions de travail
  • Avantages sociaux
